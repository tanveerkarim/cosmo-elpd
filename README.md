# Cosmo-ELPD: Model Comparison & Systematics 

## Overview
This repository contains the analysis pipelines for evaluating cosmological model performance using Expected Log Predictive Density (ELPD) and Pareto Smoothed Importance Sampling (PSIS). 

Primary applications include Leave-One-Out Cross-Validation (LOO-CV) on joint cosmology analyses, specifically: DESI DR2 BAO, Planck CMB (CamSpec) and SNIa. The goal is to understand how ELPD changes as a function of redshift and pivot redshift.
